API INFORMATION.

http://gen-review.net/api/v1.0/similar_apps_count

input {'target_id': 'com.appadvisory.drawclimber'}

returns {'similar_apps_count': '1'}

Counts how many apps are similar to the given one. Similar apps - source of generated reviews

http://gen-review.net/api/v1.0/request_file_id

input {'user_id': 'your_id'}

returns {'file_id': 'your_id 2020-02-27 22:53:44.456785'}

Returns the name of the file with generated reviews

http://gen-review.net/api/v1.0/gen

input {'user_id': 'your_id', 'file_id':'your_id 2020-02-27 22:53:44.456785', 'target_id':'com.appadvisory.drawclimber', 'lang':'en_US', 'reviews_amount':'20', 'length': 'Long', 'ratings':'1','similar_ids':['com.Comp1.App1', 'com.Comp2.App2','com.Comp3.App3']}

returns {'result': 'Completed'}

Saves reviews to a given file_id

http://gen-review.net/api/v1.0/get_reviews

input {'file_id': 'your_id 2020-02-27 22:53:44.456785','ratings':1}

returns {'1': 'Cool app', '2':'Love it', '3':'Super game!', '4':'Addictive','5':'Like it very much','6':'Super amazing'}

Returns generated reviews from file_id with specified rating (ratings = -1 negative reviews, ratings = 0 3 rate reviews, ratings = 1 positive reviews)

'target_id' - (string) id of the app you want to generate reviews for.
Example: 'com.Company.App'
'similar_apps_count' - (string) number of sources to get reviews from.
Example: '3'
'user_id' - (string) your id given by the API developer. Used for authorization.
Example: 'dfhjnlt74jl'
'file_id' - (string) name of the file where your reviews are stored.
'lang' - (string) language you want to get reviews in.
Example: 'en_US'
'reviews_amount' - (string) minimum amount of reviews you want to generate.
Example: '500'
'length' - (string) 'Long' or 'Short'. 'Long' - generate long reviews. 'Short' - generate short reviews.
'ratings' - (int) 1 or 0 or -1. 1 = generate positive reviews. 0 = generate reviews with rate 3. -1 = generate negative reviews.
'similar_ids' - (array) id's of the apps you want to get reviews from.
If similar_apps_count function returned low amount of similar apps (1-2), you can set similar apps manually.
If there are enough similar app returned by similar_apps_count, then you can skip this parameter.
Example: ['com.Comp1.App1', 'com.Comp2.App2','com.Comp3.App3']
'result' - (string) 'Completed'. Returns when the generator finishes its work.
